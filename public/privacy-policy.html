KOLFlow 隐私政策

更新并生效日期：2026 年 7 月 22 日

KOLFlow 是自托管开源软件。您的账号与业务数据通常保存在您所连接的 KOLFlow 实例中；该实例的部署者或管理员是实际的数据处理者。请在使用前同时确认该部署者公示的主体信息与隐私说明。

本政策说明 KOLFlow 各项业务功能处理个人信息的目的、方式、范围，以及您查询、更正、导出和删除信息的途径。 登录或注册前，请您完整阅读并确认理解本政策。

一、运营者基本情况与联系方式

  • 软件开发者与发布者：熊猫不是猫QAQ（个人开发者，非企业主体）。
  • 注册/办公地址：个人开发者远程办公，无企业注册地址或固定对外办公场所。
  • 个人信息保护负责人及联系邮箱：676096193@qq.com。
  • 投诉、举报与问题反馈：发送邮件，或通过 GitHub Issues 提交并跟踪处理进度。
  • 自托管实例运营者：若 KOLFlow 由其他组织或个人部署，实际运营主体、办公地址和联系方式以该实例部署者公示的信息为准。

二、基本业务功能、扩展业务功能与必要信息

基本业务功能是账号注册、登录及身份认证。完成这些功能需要邮箱和密码； 注册时还需要部署者配置的邀请码。拒绝提供必要信息将无法创建账号或登录，但不影响您查看本政策。

扩展业务功能包括个人资料、商单、品牌联系人、账单、资产、待办、评论、数据导入导出、 WebDAV 同步和外部 API。相关字段由您按需主动填写或配置，除界面明确标注的业务必填项外均属于非必要信息； 不使用某项扩展功能，不影响账号登录和其他独立功能。

五、信息的存储、传输与安全

账号和业务数据通常存储在当前 KOLFlow 实例连接的服务器中。网页端和 Android APP 均保留 HTTP 与 HTTPS 连接方式； 登录、注册、修改邮箱或密码以及账号注销的敏感提交载荷使用 RSA-OAEP-256 与 AES-256-GCM 混合加密， 避免邮箱、密码和邀请码直接出现在 HTTP 请求体中，并使用一次性挑战值防止密文重放。 KOLFlow 还采用密码哈希、访问令牌、接口鉴权、请求限流和安全响应头等措施降低风险。

HTTP 页面无法抵御主动中间人篡改，HTTPS 仍是推荐部署方式。自托管部署者应配置有效 TLS 证书、及时更新软件、限制服务器访问并妥善保管数据库和备份。 任何网络服务都无法保证绝对安全；如发生疑似泄露，请立即停用相关令牌并联系实例管理员。

六、您的个人信息权利与账号注销

  • 查询与更正：可在“设置－个人资料/账号安全”中查看并修改账号资料。
  • 导出：可在“设置－数据管理”中导出当前账号的业务数据备份。
  • 删除单项数据：可在相应业务页面删除记录，或在“数据管理”中清空业务数据。
  • 注销账号：进入“设置－账号安全－注销账号”，输入当前密码并二次确认。完成后，当前实例会立即删除账号及其商单、品牌、账单、待办、资产、评论、日志和设置等关联数据，且无法恢复。
  • 撤回可选授权：可停用 WebDAV、清除本地配置或重新生成 API Key。撤回不影响此前依法或依约完成的处理。

注销当前账号不会自动删除您已同步到 WebDAV、已导出的备份或已交给其他工具的数据； 请同时到对应服务中删除。若页面功能无法使用，可联系实例管理员或通过本政策第一节所列渠道提交申请。

七、未成年人保护

KOLFlow 面向具备相应民事行为能力的内容创作者和团队，不专门面向未成年人提供服务。 未成年人应在监护人同意和指导下使用；如发现未经适当同意处理了未成年人信息，请通过本政策所列渠道联系我们。

八、投诉、举报与争议处理

您可以发送邮件至 676096193@qq.com，或通过 GitHub Issues 提交投诉、举报、隐私请求或安全问题。请勿在公开 Issue 中粘贴密码、令牌、身份证件或完整业务数据。 我们将在核验必要信息后尽快处理；与具体自托管实例有关的数据请求，应优先联系该实例部署者。

本政策可能随功能、部署方式或信息处理方式变化而更新。更新后的版本将在本页面标注日期； 涉及个人信息处理规则的重要变化，应由软件发布者或实例管理员以适当方式提示。
